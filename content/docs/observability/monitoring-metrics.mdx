llm-d exposes a rich set of metrics through Prometheus — covering inference queue depth, KV cache utilization, TTFT distributions, and per-replica throughput. This page describes the key metrics, how to install the monitoring stack, and example Grafana dashboards for visualizing llm-d's performance.
