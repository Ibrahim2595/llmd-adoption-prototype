llm-d supports distributed tracing via OpenTelemetry, propagating trace context across vLLM model servers, the routing proxy, and the Endpoint Picker (EPP) component. This page explains how to configure OpenTelemetry tracing, export traces to Jaeger or other backends, and interpret traces for latency debugging.
