The gateway-free configuration is the fastest way to get llm-d running. You deploy a vLLM model server with the llm-d scheduler and KV cache sidecar injected automatically. Requests go directly to the model server — no gateway hop, no routing policy overhead.

This is ideal for single-model deployments, development environments, and benchmarking baselines.

<StepGuide>
  <Step number={1} title="Add the llm-d Helm repository">
    Add and update the Helm chart repository before installing any components.

    ```shell
    helm repo add llm-d https://llm-d.github.io/helm-charts
    helm repo update
    ```
  </Step>

  <Step number={2} title="Install the llm-d operator">
    The operator manages CRDs and injects sidecars into model server pods. Install it cluster-wide:

    ```shell
    helm install llm-d-operator llm-d/llm-d-operator \
      --namespace llm-d \
      --create-namespace \
      --version 0.5.0
    ```

    Wait for the operator pod to reach `Running` status before continuing:

    ```shell
    kubectl rollout status deployment/llm-d-operator -n llm-d
    ```
  </Step>

  <Step number={3} title="Deploy a model server">
    Create a `ModelServer` custom resource. This example deploys `Qwen/Qwen3-14B` on a single H100:

    ```yaml
    apiVersion: inference.llm-d.ai/v1alpha1
    kind: ModelServer
    metadata:
      name: qwen3-14b
      namespace: llm-d
    spec:
      model: Qwen/Qwen3-14B
      replicas: 1
      resources:
        limits:
          nvidia.com/gpu: "1"
      engine:
        type: vllm
        version: "0.5.0"
      cache:
        enabled: true
        tiered: false
    ```

    Apply it:

    ```shell
    kubectl apply -f modelserver.yaml
    ```
  </Step>

  <Step number={4} title="Send your first request" last>
    Port-forward the model server service and send a test completion:

    <CodeTabs labels="curl,Python">
      ```shell
      kubectl port-forward svc/qwen3-14b 8000:8000 -n llm-d &

      curl http://localhost:8000/v1/completions \
        -H "Content-Type: application/json" \
        -d '{
          "model": "Qwen/Qwen3-14B",
          "prompt": "Kubernetes is",
          "max_tokens": 64
        }'
      ```

      ```python
      from openai import OpenAI

      client = OpenAI(base_url="http://localhost:8000/v1", api_key="unused")

      response = client.completions.create(
          model="Qwen/Qwen3-14B",
          prompt="Kubernetes is",
          max_tokens=64,
      )
      print(response.choices[0].text)
      ```
    </CodeTabs>
  </Step>
</StepGuide>
