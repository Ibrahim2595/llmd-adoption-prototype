---
title: "Latency Predictor"
description: "The latency predictor model used for scheduling decisions."
sidebar_order: 5
---

The latency predictor is a lightweight model that estimates the expected end-to-end latency for a given request before it is dispatched, enabling the inference scheduler to make smarter routing decisions that minimize tail latency and prevent individual replicas from becoming bottlenecks.
