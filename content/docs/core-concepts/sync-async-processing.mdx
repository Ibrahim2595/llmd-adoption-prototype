---
title: "Synchronous and Asynchronous Processing"
description: "The difference between synchronous and asynchronous inference modes in llm-d."
sidebar_order: 6
---

llm-d supports both synchronous and asynchronous inference modes to accommodate workloads with different latency requirements. Synchronous mode delivers results inline with the request, while asynchronous mode decouples submission from execution using a queue-based architecture optimized for throughput-sensitive or latency-tolerant jobs.
