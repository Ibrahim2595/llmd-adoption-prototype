title: "Autoscaling"
description: "How llm-d autoscales model server replicas based on load."
sidebar_order: 3

llm-d provides SLO-aware autoscaling strategies that react to inference-specific signals such as queue depth, KV cache utilization, and in-flight request counts — rather than lagging indicators like GPU utilization. This page explains the autoscaling architecture and the different strategies available for homogeneous and heterogeneous deployments.
