---
title: "Tiered KV Cache Offloading"
description: "Offloading KV cache to CPU or remote storage to improve cache hit rates."
sidebar_order: 4
---

llm-d extends the KV cache beyond GPU HBM into a configurable tiered hierarchy — CPU RAM, local NVMe, and remote shared storage — so that frequently reused prompt prefixes remain warm and avoid redundant recomputation. This page describes the tiering architecture, supported backends, and guidance for selecting the right tier configuration for your workload.
