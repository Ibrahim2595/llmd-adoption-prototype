# llm-d Website

The new website for llm-d — an open-source Kubernetes-native distributed LLM inference framework.

> **Note:** This is a prototype for exploring the llm-d adoption strategy. It is not the production site yet. We welcome feedback and critiques — please share them in #sig-docs on the llm-d Slack community or reach out to Ibrahim Ibrahim directly.

## Prerequisites

- Node.js 18+
- pnpm

## Getting started

```bash
pnpm install
pnpm dev
```

Open http://localhost:3000.

## Contributing content

- **Docs:** add `.mdx` files to `content/docs/`
- **Blog:** add `.mdx` files to `content/blog/`
- **Videos:** add entries to `content/videos.yaml`

See [CLAUDE.md](./CLAUDE.md) for full project conventions.
